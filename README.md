# Real-Time Electricity Monitoring Dashboard (Simulated)

This Python terminal dashboard simulates real-time electricity monitoring for multiple circuits.

- Circuits: Living Room, Kitchen, Server Room, Office, Garage  
- Color-coded readings: Green ≤7A, Yellow 7–12A, Red >12A  
- Updates every 2 seconds  

## How to Run

1. Clone the repo
2. Navigate to the folder in terminal
3. Run:
```bash
python3 monitor_simulation.py
